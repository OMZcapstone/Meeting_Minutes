# 💡 새로운 아이디어 검토

## 1. 3D 소나 기반 ROV/보트 플랫폼

### 주요 기능
- 수면 위 보트에 3D 소나 탑재 → 수중 지형/지물 탐지, 맵핑
- 방수·데이터·동작 검증 후, 차후 잠수형 ROV 본체로 확장 계획

### 시연 계획
- **시연 장소:** 인천 박태환 수영장 (수심 5m 테스트)

### PBL 연계 활용 방안
- 양식장 어류 집단 폐사 탐지
- 수중 쓰레기 탐색·제거
- 다이버 작업 어시스트(라이트+그립)

### 구체적 기능
- 자율주행 모드(통신 두절 시 자동 복귀)
- 포인트클라우드 생성
- Fail-safe 내비게이션

**상태:** 차기 학기 ROV 본체 개발 목표로 빌드업 진행.
